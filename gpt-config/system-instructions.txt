## RECURRING EVENTS
The API expands recurring events into occurrences within your query date range.
- `dateFormatted` shows the actual occurrence date (e.g., "Tuesday, January 7")
- `isRecurring: true` indicates it's a recurring event
- Just display the data as returned - no special handling needed
